"Football league standing"

Projekt Football League Tournament to projekt, która pozwala użytkownikom zarządzać drużynami piłkarskimi, zawodnikami, meczami i tabelami. Projekt zapewnia użytkownikom łatwy w użyciu interfejs do zarządzania całym turniejem, od rejestracji drużyn i zawodników po planowanie meczów, zapisywanie wyników meczów i aktualizację tabeli.

  1. Zarządzanie drużyną i zawodnikami: Projekt umożliwia użytkownikom tworzenie i zarządzanie drużynami piłkarskimi oraz zawodnikami. Użytkownik może stworzyć nową drużynę, podając podstawowe informacje, takie jak nazwa drużyny, miasto, kraj, stadion, trener, lista zawodników. Użytkownik może również dodawać, aktualizować lub usuwać graczy w drużynie.
  2. Planowanie i zarządzanie meczami: Projekt umożliwia użytkownikom planowanie i zarządzanie meczami piłki nożnej. Użytkownik może zaplanować nowy mecz, podając drużynę gospodarzy, drużynę gości, datę meczu, godzinę i lokalizację. Po rozegraniu meczu użytkownik może wprowadzić wyniki meczu, w tym liczbę bramek zdobytych przez każdą drużynę.
  3. Tabele i rankingi: Projekt zapewnia użytkownikom w czasie rzeczywistym tabele i rankingi drużyn na podstawie ich wyników w turnieju. Klasyfikacja jest aktualizowana automatycznie po każdym rozegranym meczu i obejmuje liczbę rozegranych meczów, wygrane, remisy, przegrane, bramki zdobyte, bramki stracone, różnicę bramek i punkty.

Schematy :


Football Player : zawodnik

  • ID : numer unikalny
  • Name : imię i nazwisko zawodnika
  • Position : pozycja zawodnika (defender, )
  • Jersey Number : numer w drużynie, jezeli ma drużyna
  • Nationality : kraj pochodzenia zawodnika
  • Age : wiek zawodnika
  • Height : wzrost zawodnika
  • Weight : waga zawodnika
  • Club : ID druzyny, jezeli niem pusty

Football Team : drużyna

  • ID : numer unikalny
  • Name : nazwa
  • City : miasto
  • Country : kraj drużyny
  • Stadium : stadium
  • Coach : imię i nazwisko trenera
  • Players : tablica z zawodnikami drużyny

Football Turniej :

  • ID : numer unikalny
  • Name : nazwa turneju
  • Date Start : data poczatku turnieju
  • Date End : data konca turnieju
  • Teams : tablica z drużynami

Football Standings :

  • Tournament : numer unikalny turnieju (ID)
  • Team : nazwa drużyny
  • Matches Played : ile ma zegranych meczy
  • Wins : wygranych
  • Draws : remisy
  • Losses : przegranych
  • Goals For : pilek trafili
  • Goals Against : pilek przepuscili
  • Points : zdobyte punkty

Football Match :

  • Tournament : numer unikalny turnieju (ID)
  • Home Team : drużyna domowa (ID)
  • Away Team : drużyna gosci (ID)
  • Home Goals : pilki drużyna domowej
  • Away Goals : pilki drużyna gosci
  • Additional Time : dodatkowy czas byl grany
  • Date : data
  • Time : czas

Functions


Zawodniki

  • add_player() : umożliwia dodanie nowego zawodnika
  • update_player() : pozwala edytować szczegóły istniejąca zawodnika
  • delete_player(id) : pozwala usunąć istniejąca zawodnika
  • get_player(id) : pozwala dostac informacja o zawodnikie
  • get_players() : lista wszystkich zawodnikow

Drużyny

  • add_team : umożliwia dodanie nowej drużyny
  • update_team : pozwala edytować szczegóły istniejącej drużyny
  • delete_team(id) : pozwala usunąć istniejąca drużyna
  • get_team(id) : pozwala dostac informacja o drużynie
  • get_teams : lista wszystkich drużyn

Turniej

  • add_tournament() : umożliwia dodanie nowego turnieju
  • update_tournament() : pozwala edytować szczegóły istniejąca turnieju
  • delete_tournament(id) : pozwala usunąć istniejący turniej
  • get_tournament(id) : pozwala dostac informacja o turnieju
  • get_tournament_standing(id) : pozwala dostac klasyfikacja turnieju piłki nożnej

Meczy

  • add_match() : umożliwia dodanie nowego meczu
  • update_match() : pozwala edytować szczegóły istniejącego meczu
  • delete_match(id) : pozwala usunąć istniejący mecz
  • get_match(id) : pozwala dostac informacja o meczu
  • get_matches() : lista wszystkich meczow
